INSTALLATION
------------
<h3>First create a database called "redbridge" </h3>

1) just after pulling from git
~~~
Compoer install
~~~

2) to create tables and  populate them (50 sample books will be filled)
~~~
yii migrate
~~~

3) to run tests
~~~
./vendor/bin/codecept run unit
~~~

4) To see swagger endpoints
~~~
localhost/swagger
~~~


REQUIREMENTS
------------
<h3>php 8.1 </h3>
<h3>PostGreSql 14 </h3>
